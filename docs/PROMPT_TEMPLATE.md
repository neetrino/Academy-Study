# Шаблон промпта для AI

**Курс:** AI-Coding в Cursor  
**День 1:** Правильные промпты — ТЗ → план → код

---

## Структура промпта

```
[Контекст] — что уже есть, какой проект, стек
[Цель] — что нужно сделать
[Ограничения] — что нельзя трогать, стек, стиль
[Критерии] — как понять, что готово
[Шаги] — если нужно пошагово (опционально)
[Риски] — что может сломаться (опционально)
```

---

## Шаблон (копируй и заполняй)

```
## Контекст
- Проект: [название]
- Стек: [языки, фреймворки]
- Связанные файлы: [@file1, @file2]

## Цель
[Одно предложение: что нужно сделать]

## Ограничения
- Не менять: [что не трогать]
- Использовать: [стек, паттерны]

## Критерии готовности
- [ ] [Критерий 1]
- [ ] [Критерий 2]

## Риски (если есть)
- [Что может сломаться]
```

---

## Примеры

### Плохой промпт
> Сделай красивую кнопку.

### Хороший промпт
> ## Контекст
> - Проект: учебное приложение на React + TypeScript
> - Компоненты: PascalCase, стили через CSS modules
>
> ## Цель
> Добавить кнопку «Отправить» в форму контакта.
>
> ## Ограничения
> - Использовать существующий Button из ui/
> - Не менять layout формы
>
> ## Критерии
> - Кнопка внизу формы
> - Disabled, пока форма невалидна
> - При клике вызывается onSubmit
