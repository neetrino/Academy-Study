[Почему нужно было принять решение.]

## Решение

- `.env` для локальной разработки, в `.gitignore`
- В проде — переменные окружения
- `.env.example` с placeholder'ами в репо

## Последствия

- Плюс: секреты не в коде
- Минус: настройка env на каждом окружении
