# Неделя 1 — Путеводитель студента

**Курс:** AI-Coding в Cursor (VibeCoding → Production)  
**Тема:** База AI-coding — навыки, без которых дальше только боль

---

## Что ты изучишь

1. **Промпты** — как писать задания AI как инженер
2. **Chat vs Agent** — правильный режим под задачу
3. **Контекст** — что давать AI, что не надо
4. **Рефакторинг** — без разрушений, с тестами
5. **Debugging** — через гипотезы, с обоснованием
