# AI-Coding в Cursor

**VibeCoding → Production** — учебный план для программистов, работающих с Cursor.

---

## О курсе

Превратить разработчика в специалиста, который умеет **управлять AI**, а не «просить кодик». Результат: фичи, тесты, PR, документирование — без поломок продакшена «потому что AI так сказал».

| Параметр | Описание |
|----------|----------|
| **Формат** | 4 недели, 5 дней/неделю, 2–4 ч/день |
| **Артефакты** | Мини-проект, `.cursor/rules`, промпты, ADR |

---

## Начало

### Неделя 0 (день 0–1)

1. Открой **[WEEK_0_PLAN/README_STUDENT.md](WEEK_0_PLAN/README_STUDENT.md)** — путеводитель
2. Выполни Урок 0.1 и 0.2 по [WEEK_0_PLAN/WEEK_0_PLAN.md](WEEK_0_PLAN/WEEK_0_PLAN.md)
3. Изучи [docs/AI_WORKFLOW.md](docs/AI_WORKFLOW.md) — правила работы с AI

### Неделя 1 (день 1–5)

1. Открой **[WEEK_1_PLAN/README_STUDENT.md](WEEK_1_PLAN/README_STUDENT.md)** — путеводитель
2. Выполни Дни 1–5 по [WEEK_1_PLAN/WEEK_1_PLAN.md](WEEK_1_PLAN/WEEK_1_PLAN.md)
3. Создай артефакты в `docs/`: PROMPT_TEMPLATE, CHAT_VS_AGENT_CHECKLIST, CONTEXT_PACKAGES, BUG_REPORT_TEMPLATE

### Структура проекта

```
├── README.md               ← ты здесь

├── docs/
```
